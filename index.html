CSS Transform

translate(x, y)

translate

translate-сдвигает элемент на новое место,перемещая относительно изначального положения объекта вправо и вниз,используя координаты X и Y. Записывается следующим образом translate(x,y);

scale (x,y)

scale

scale-масштабирует элементы, делая их больше или меньше. По умолчанию значения указанны как еденица(1)-это реальный размер объекта.Первое значение масштабирует элемент по ширине,второе-по высоте.Отрицательные значения отображают элемент зеркально.

rotate (угол+deg)

rotate

rotate-поварачивает элементы на заданное кооличество градусов,отрицательные значения поварачивают элемент против часовой сттрелки,положительные- по часовой.Записывается значения как градус поворота и deg.

skew (x-угол+deg, y-угол+deg)

skew

skew- деформирует (искажает) стороны объекта относительно координат осей.Если указано одно значение, второе будет определенно браузером автоматически.Записываются значения как градус поворота и deg.

matrix (a,b,c,d,x,y)

matrix

matrix (a,b,c,d,x,y) - позволяет объединять несколько функций 2D-трансформаций в одной.В качестве трансформации допустимы поворот,масштабирование,наклон(деформация) и изменение положения. Свойства указываются без едениц измерения.

Множественный 2D трансформации

translate
scale
rotate

Так же,подобного можно добиться используя множественные трансформации.А именно записать несколько значений в одно свойство transform.

transform-origin

transform origin

transform-origin-позволяет сместить центр трансформации,относительно которого происходит изменение положения/размера/формы элемента.

prespective, perspective-origin

perspective,
perspective-origin

perspective,perspective-origin-установка глубины и смена точки начала координат

translate3D(x,y,z)

translate3D

translate3D-по анологии со своим 2D братом это значение задает перемещение элемента,но,в 3D-пространстве.Помимо оси X и Y добвляется ось Z.

scale3D(x,y,z)

scale3D

scale3D-задаёт трехмерное масштабирование по одной из осей X,Y,Z.

rotate3d(x,y,z,deg)

rotate3D

rotate3D-вращает элемент относительно трех осей.Элемент поворачивается под углом,задаваемым последним параметром относительно вектора направления X,Y,Z.

matrix3d(n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n)

matrix3D

matrix3D-задает трехмерное преобразование как однородную матрицу размером 4x4 с шестнадцатью значениями в столбце.
